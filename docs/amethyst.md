# Amethyst Commands & Notes

## Layouts

- Tall
- Tall Right
- Two Pane
- Fullscreen

## Commands

Modifiers

```
a+s    mod1
c+a+s  mod2
```

Controls
```
mod1+A     Select Tall layout
mod1+S     Select Binary Space Partition layout
mod1+D     Select Wide layout
mod1+F     Select Full Screen layout
mod2+A     Select Tall Right layout 
mod2+D     Select Two-Pane layout

mod2+Z     Relaunch Amethyst

mod1+,     Increase main pane count
mod1+.     Decrease main pane count

mod1+J     Move focus counterclockwise
mod1+K     Move focus clockwise
mod1+M     Move focus to main window

mod2+J     Swap focused window counter clockwise
mod2+K     Swap focused window clockwise
mod1+<cr>  Swap focused window with main window

mod1+F     Toggle float for focused window
----------
mod2+X     Toggle focus follows mouse

m1+space   Cycle layout forward
m2+space   Cycle layout backwards

mod1+H     Shrink main pane
mod1+L     Expand main pane


mod1+Z     Force windows to be re-evaluated

mod1+I     Display current layout

mod1+B     Select Tall Right layout
mod1+T     Select Two Pane layout
```
